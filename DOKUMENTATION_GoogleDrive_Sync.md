# Übergabeprotokoll: Blog-Artikel erstellen mit Google Docs

---

## 1. Funktionsweise

Du schreibst deine Artikel in **Google Docs** und importierst sie als HTML-Datei in den Blog.

### Vorteile
- Schreiben in gewohnter Google Docs Umgebung
- Keine externen APIs oder Keys nötig
- Funktioniert komplett offline
- Formatierungen bleiben erhalten

---

## 2. So erstellst du einen neuen Blogartikel

### Schritt 1: Artikel in Google Docs schreiben
1. Öffne Google Drive
2. Erstelle ein neues Dokument
3. Schreibe deinen Artikel mit Überschriften (Überschrift 1, 2, 3)
4. Nutze Fett, Kursiv, Listen etc. - alles wird übernommen

### Schritt 2: Als HTML exportieren
1. In Google Docs: Menü **Datei**
2. Klick auf **Herunterladen**
3. Wähle **Webseite (.html)**
4. Die Datei wird auf deinen Computer gespeichert

### Schritt 3: Im Admin importieren
1. Gehe zu **Admin → Blog**
2. Klick auf **Neuen Artikel schreiben**
3. Fülle Titel, Kurzbeschreibung etc. aus
4. Klick auf den Button **"HTML-Datei aus Google Docs hochladen"**
5. Wähle die exportierte .html Datei aus
6. Der Inhalt wird automatisch eingefügt
7. Speichern und fertig!

---

## 3. Bilder einbinden

**Bilder werden nicht automatisch importiert.** So fügst du sie manuell hinzu:

1. Lade das Bild in Google Drive hoch
2. Rechtsklick → "Öffnen mit" → "Google Drawings"
3. Klick auf das Bild → "In Zwischenablage kopieren"
4. Füge es direkt in Google Docs ein (oder als Bilddatei im Admin hochladen)

Oder:
1. Bild separat im Admin unter "Beitragsbild (URL)" einfügen
2. Bild-URL aus Google Drive kopieren (mit Freigabe-Link)

---

## 4. Bekannte Einschränkungen

| Limitation | Erklärung |
|------------|-----------|
| Keine Tabellen | Tabellen werden nicht unterstützt |
| Keine eingebetteten Medien | Videos/Audio nicht importierbar |
| Google-spezifische Styles | Werden beim Import bereinigt |

---

## 5. Tipp: Zwischenablage für Schnellimport

Für kurze Artikel kannst du auch:
1. Text in Google Docs schreiben
2. Alles markieren und kopieren (Strg+C)
3. Im Admin in das Feld "Inhalt (HTML / Text)" einfügen
4. Die Formatierungen bleiben meist erhalten
